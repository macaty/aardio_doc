thread 成员列表

thread.token()

返回对象:processTokenObject

thread.token(threadHandle,access,gc)

获取线程令牌，返回 process.token 对象。
@threadHandle 参数指定线程句柄，省略取当前线程句柄，
access指定访问权限,省略时默认值为 _TOKEN_QUERY
@gc 指定返回对象在回收时默认自动调用close函数，省略时默认为 true
