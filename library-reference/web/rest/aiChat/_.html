web.rest.aiChat 库模块帮助文档

web.rest 成员列表

web.rest.aiChat

用于调用 Anthropic 或 OpenAI 兼容 AI 聊天接口。
范例

web.rest.aiChat()

返回对象:webRestAiChatObject

web.rest.aiChat(config)

web.rest.aiChat(  
    proxy = proxy,  
    model = "@claude-3-5-sonnet-latest",  
    temperature = 0.1,  
    maxTokens = 1024,  
    url = ""  
)/*创建 AI 聊天客户端。参数说明：  
url 指定 Anthropic 或 OpenAI 兼容接口网址。  
model 指定模型名称，首字符为 @ 则使用 Anthropic 接口。  
可选用 proxy 指定代理服务器，代理格式: https://www.aardio.com/zh-cn/doc/library-guide/std/inet/proxy   
temperature 指定温度。  
maxTokens 限定最大回复长度。*/

webRestAiChatObject 成员列表

webRestAiChatObject.messages(msg,writeDelta)

调用聊天会话接口。
msg 参数指定 web.rest.aiChat.message 对象。
writeDelta 指定输出 AI 回复文本的回调函数，
回调参数为文本时则应输出增量回复，回调参数为 null 时完成输出。
