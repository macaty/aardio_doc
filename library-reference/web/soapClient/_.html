web.soapClient 库模块帮助文档

web 成员列表

web.soapClient

SOAP客户端支持库

web.soapClient()

返回对象:webSoapClientObject

web.soapClient(WebService网址,用户名,密码)

创建SOAP客户端,
必须指定有效WSDL网址,
用户名与密码为可选参数

webSoapClientObject 成员列表

webSoapClientObject.bindService()

绑定服务,参数指定服务名,无参数绑定默认服务
在创建对象时此函数会被自动调用一次并绑定默认服务

webSoapClientObject.http

用于发送请求的http客户端

返回对象:inetHttpObject

webSoapClientObject.soapPortName

当前绑定接口名

webSoapClientObject.soapServiceName

当前绑定服务名

webSoapClientObject.soapTargetNamespace

SOAP目标名字空间

webSoapClientObject.soapUrl

发送SOAP请求的目标URL，默认自WSDL中自动获取

webSoapClientObject.接口函数名字(调用参数)

!web_soapClient.lastResultXml = 调用接口最后一次返回值创建XML节点对象

返回对象:xnodeObject
