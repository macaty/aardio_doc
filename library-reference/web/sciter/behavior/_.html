web.sciter.behavior 库模块帮助文档

web.sciter.behavior 成员列表

web.sciter.behavior.collapsibleByIcon

折叠按钮(图标)

web.sciter.behavior.collapsibleList

折叠列表(项目可展开单项或全部折叠)

web.sciter.behavior.dragable

使节点可拖动
样式中可使用 -drag-margin:"上 右 下 左"指定拖动边距属性以限制拖动范围
不限制拖动范围则可以将节点拖出窗口,并触发onPopup***系列事件
节点可使用dragable属性设置是否允许在该节点开始拖动

web.sciter.behavior.dropdown

下拉选框

web.sciter.behavior.expandableList

扩展列表(实现手风琴效果/Outlook风格菜单)

web.sciter.behavior.grid

网格控件

web.sciter.behavior.gripper

用于创建拖动手柄,
拖动该对象时拖动父节点

web.sciter.behavior.lightBoxDialog

高亮对话框效果
使用 import 语句在加载页面前导入即可生效

web.sciter.behavior.path

文件路径显示支持
超出显示宽度显示省略号,鼠标悬停提示完整内容

web.sciter.behavior.popup

弹出面板

web.sciter.behavior.scroller

自动滚动效果

web.sciter.behavior.selectionBox

鼠标选框

web.sciter.behavior.shellIcon

文件图标效果
使用 import 语句在加载页面前导入即可生效

web.sciter.behavior.sizer

使对象可通过鼠标拖动调整大小

web.sciter.behavior.sortableGrid

排序网格控件

web.sciter.behavior.splitter

拆分条

web.sciter.behavior.tabs

Sciter tabs behavior
使用 import 语句在加载页面前导入即可生效

web.sciter.behavior.windowCommand

支持窗口命令,如最大化,最小化,关闭等等

web.sciter.behavior.windowSizer

用于无边框窗口以支持窗体大小可调边框
