# ttf2eot 库模块帮助文?
## ttf2eot 成员列表

### ttf2eot.convert("ttf字体名称","eot格式输出目录")

TTF字体转换为EOT格式

仅支持TTF格式

### ttf2eot.convert("ttf文件路径","eot格式输出路径")

TTF字体转换为EOT格式

省略eot路径参数则在当前目录输出同名eot后缀文件
