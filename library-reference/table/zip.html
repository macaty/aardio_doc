table 成员列表

table.zip

链接数组，需导入

table.zip(...)

参数指定多个数组时，
将所有数组相同索引的元素链接为数组，并返回包含这些数组的数组。

如果仅指定 1 个数组，
则将数组中的所有成员数组作为参数进行链接，
所有成员数组相同索引的元素链接为新的数组，
返回包含这些数组的数组

此函数至少需要一个参数，否则会抛出异常

table.zip(true,keys,values)

返回一个新的表，
以 @keys 中元素为键，以 @values 中相同索引的数组元素为值
