# fonts 库模块帮助文?
必读

可在aardio标准库、用户库视图"/fonts"目录上右键菜单中点“粘贴文?..?aardio会在 fonts 名字空间自动生成字体加载库（如果库已经存在则忽略不操作）?

1、标准库图标字体资源目录 '~/lib/fonts/.res/'
2、工程用户库图标字体资源目录 '/lib/fonts/.res/'
