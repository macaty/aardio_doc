raw.memory.allocPointer()

返回对象:rawMemoryObject

raw.memory.allocPointer(内存大小,填充数据)

分配内存并返回指针
无需lock即可直接使用指针,
可选使用结构体或字符串等指定默认填充数据
指定默认值时可以省略参数@1
可选使用参数3指定是否自动析构,默认值为true
