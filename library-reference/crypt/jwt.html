crypt.jwt 库模块帮助文档

crypt 成员列表

crypt.jwt

JSON Web Token 令牌

crypt.jwt()

返回对象:cryptJwtObject

crypt.jwt(密钥,超时)

使用字符串指定签名使用的密钥,
参数@2指定令牌超时秒数

cryptJwtObject 成员列表

cryptJwtObject.decode()

解码并返回令牌信息

cryptJwtObject.encode()

生成令牌，
参数指定要存储到客户端的数据，
参数必须是表对象

cryptJwtObject.verify()

验证参数指定的token是否在有效期内，以及签名是否有效
如果验证token有效，就解码返回令牌信息
