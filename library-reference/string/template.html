string.template

字符串模板

string.template()

返回对象:stringTemplateObject

string.template(模板字符串,模板匹配规则)

可选使用模式匹配语法指定参数匹配规则
默认匹配规则为"\${(.+?)}",匹配形如 ${参数名} 的参数
