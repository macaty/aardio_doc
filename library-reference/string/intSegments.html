string.intSegments()

解析参数@1指定的整数列表,
并返回数值从小到大排序后的数组,

参数@1可以指定单个数值,或用文本指定多个数值,
文本中数值用逗号分隔,允许用 - 指定数值区间,
例如："6881-6889,6999",
参数@1传入非文本非数值参数返回 null

string.intSegments 成员列表

用于解析文本格式的整数列表,支持区间表示法

string.intSegments.each(str)
