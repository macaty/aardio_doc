## process 成员列表


<a id="process.chm"></a>
### process.chm 
 用于打开 CHM 帮助文档。

<a id="process.chm"></a>
### process.chm(chmPath,htmlPath) 
 打开 CHM 帮助文档。  
@chmPath 指定 chm 文件路径，省略则默认打开 aardio 文档。  
@htmlPath 指定页面路径

<a id="process.chm"></a>
### process.chm(chmUrl,htmlPath) 
 打开 chm:// 协议帮助文档。
