## coroutine 成员列表


<a id="coroutine.main"></a>
### coroutine.main 
 主函数

<a id="coroutine.run"></a>
### coroutine.run(主函数, ) 
 将函数设为协程主函数,并启动协同程序.  
可在此传入不定个参数给主函数

<a id="coroutine.transfer"></a>
### coroutine.transfer(协程对象,) 
 把控制权及附加的不定个参数分派给协程对象  
省略参数@1则切换到主函数
