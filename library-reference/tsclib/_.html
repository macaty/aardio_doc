tsclib.qrcode(x,y,eccLevel,width, mode, rotation,model,mask,dataString)

打印二维码
x QRCODE条码左上角X座标
y QRCODE条码左上角Y座标
eccLevel 错误纠正能力等级
L 7%
M 15%
Q 25%
H 30%
width 1~10
mode 自动生成编码/手动生成编码
A Auto
M Manual
rotation 顺时针旋转角度
0 不旋转
90 顺时针旋转90度
180 顺时针旋转180度
270 顺时针旋转270度
model 条码生成样式
M1 (预设), 原始版本
M2 扩大版本
mask 范围：S0~S8，预设S7
dataString 条码资料内容
