comSmtpObject.password

发送邮件密码

comSmtpObject.port

SMTP 端口,默认明文发送端口 25，
使用 SSL 端口 465 请同时指定 ssl 属性为 true，
使用 TLS 端口 587 请同时指定 tls 属性为 true

comSmtpObject.reply

comSmtpObject.send()

comSmtpObject.sender

sender@example.com 或者 "Sender" <sender@example.com>

comSmtpObject.sendusing
